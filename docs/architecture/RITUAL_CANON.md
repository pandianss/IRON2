# The IRON Ritual Canon

**Layer: Civil / Social**
**Status: DRAFT**
**Dependency: Roles Charter**

Rituals are the "Stateful Moments" of the institution. They are not merely UI animations; they are transactions that change the social status of a Participant.

## I. Rites of Consistency

### 1. The Entry Rite (Oath of Service)
*The moment of joining.*
- **Trigger**: Account Creation + First Goal Setting.
- **Action**: User must read and "Sign" the `USER_COVENANT`.
- **System Effect**: `onboarded` state active. `SocialCapital` wallet created (Empty).
- **Narrative**: "You have entered the discipline."

### 2. The Momentum Mark (Elevation)
*The recognition of resilience.*
- **Trigger**: Crossing 21, 90, 365 Days.
- **Action**:
    - System issues a `Mark`.
    - User must "Claim" it by writing a reflection.
    - 3 Witnesses must "Validate" it.
- **System Effect**: Role eligibility unlocks (e.g., 90 Days -> Mentor).
- **Narrative**: "You have built a self."

### 3. The Year Close (Renewal)
*The cycle completes.*
- **Trigger**: Day 365.
- **Action**: The `BehavioralLedger` for the year is "Sealed" (Archived).
- **Rite**: The "Burning of the Failures" (Visual removal of fractured streaks).
- **Narrative**: "The past is written. The future is clean."

---

## II. Rites of Fracture & Crisis

### 4. The Fracture Protocol (Judgment)
*The breaking of the line.*
- **Trigger**: Missed Day (after buffers).
- **Action**:
    - Screen goes monochrome.
    - No other interaction allowed until "Acknowledgement".
    - User must select "Reason" (Honesty).
- **System Effect**: `STREAK_FRACTURED`. Capital Burned.
- **Narrative**: "Reality is acknowledged."

### 5. The Appeal (Petition)
*The plea for mercy.*
- **Trigger**: User disputes a Fracture.
- **Action**:
    - User stakes `SocialCapital`.
    - User summons `Chronicler` (optional) to add context.
    - Case sent to `Steward` Queue.
- **System Effect**: State frozen in `APPEAL_PENDING`.

---

## III. Rites of Restoration

### 6. The Return Ceremony (Resurrection)
*The hardest step.*
- **Trigger**: Returning after >14 days Dormancy.
- **Action**:
    - User pays "Resurrection Cost" (SC or Momentum resetting).
    - "Bell Ringing": Graph notified of return.
    - Witnesses can "Welcome Back" (Small SC grant to returnee).
- **Narrative**: "The door was never locked."

### 7. The Service Oath (Authority)
*The taking of office.*
- **Trigger**: Promoting to Sentinel / Steward.
- **Action**: User accepts "Burden of Office" (Higher penalties, mandatory review duties).
- **System Effect**: Admin permissions unlocked.
- **Narrative**: "Power is a burden."
