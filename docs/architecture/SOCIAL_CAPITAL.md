# The Social Capital Constitution

Authority: Governance Layer (Social Economy)
Status: DRAFT

## 1. Definition
**Social Capital (SC)** is the currency of trust and service within IRON. 
It is distinct from **Engagement Score** (Personal Momentum). 

> **Momentum is earned by sweat.**
> **Social Capital is earned by service.**

## 2. The Economy of Trust

### Earning SC (Minting)
| Action | Description | Value |
|--------|-------------|-------|
| **WITNESS**| Validating a partner's workout | +1 SC |
| **PACT_SAVE** | Using a freeze token to save a friend | +5 SC |
| **MENTOR** | Successfully onboarding a new user | +20 SC |
| **STREAK_MILESTONE** | Reaching 30/60/90 days (Reliability) | +10 SC |

### Losing SC (Decay & Penalties)
| Event | Description | Cost |
|-------|-------------|------|
| **FRACTURE** | Breaking a streak (Breach of Trust) | -50% SC |
| **DORMANCY** | Abrupt exit from system | -100% SC |
| **INACTIVITY** | No social interaction for 7 days | -1 SC/day |

## 3. The Power of Capital
Social Capital is not just a score. It is power.

### Spending SC
1.  **Appeal**: A user with High SC (>100) can "Appeal" a `STREAK_FRACTURED` state *once* per 90 days. The community "votes" to grant the pardon using their own SC.
2.  **Emergency Freeze**: Purchase an emergency freeze token (Cost: 50 SC).
3.  **Resurrection Disc**: Reduce Identity Debt for self or friend (Cost: 200 SC).

## 4. Invariants
- **No Buy-In**: SC cannot be purchased with money.
- **No Momentum Trade**: SC cannot buy Engagement Score or Rank. Personal fitness cannot be outsourced.
